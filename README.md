# adorable-avatars

### Includes
* Node
* Express
* Bower
  * jQuery
  * Ember
  * Handlebars
* Gulp
* Browserify
* Stylus
* Coffeescript

### Setup

1. Clone this repo
2. run `npm install`


### Running

1. Run the server: `npm start`
2. Browse to: `http://localhost:3007`


### Deploying

1. Create your heroku instance `heroku create`
2. Deploy `git push heroku master`
3. Dance.
